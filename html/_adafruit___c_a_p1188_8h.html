Detailed Description

This is a library for the CAP1188 8-Channel Capacitive Sensor

Designed specifically to work with the CAP1188 I2C/SPI 8-chan Capacitive Sensor from Adafruit

Pick one up today in the adafruit shop! ---—> https://www.adafruit.com/product/1602

These sensors use I2C/SPI to communicate, 2+ pins are required to interface

Adafruit invests time and resources providing this open source code, please support Adafruit andopen-source hardware by purchasing products from Adafruit!

Limor Fried/Ladyada (Adafruit Industries).
